<h2>🏢입사(IPSA)</h2>
  <p>입사란 공유오피스 소개 웹페이지 입니다. 
  기본적인 UI설계 및 MySql 데이터베이스와 연동하는 것을 목표로 진행했던 프로젝트입니다.</p>
  
  ![공유오피스 입사](https://user-images.githubusercontent.com/86067390/147403806-65daef3f-5b63-49c3-ac6d-1ea2d72147ff.png)

  
<h2>🗓️개발기간(9일)</h2>
  <ul>
<li>프로젝트 기획, UI 설계 : 2021.09.27 ~ 2021.10.01</li>
<li>db설계 및 기본기능 구현 : 2021.10.19 ~ 2021.10.22</li>
  </ul>
  
<h2>✨주요 기능</h2>
  <ul>
    <li>Servlet, jsp를 이용하여 구현</li>
    <li>메뉴바 왼쪽 고정</li>
    <li>카카오 맵 API를 이용하여 지도 구현</li>
    <li>회원가입, 로그인</li>
    <li>게시글 등록, 수정, 삭제</li>
    <li>상세페이지별 목록 표출</li>
  </ul>
